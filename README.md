# Smart-Tech-News-Digest

A Python-based application that fetches and filters technology news articles from TechCrunch, processes the data by category and genre, and serves it using a simple web interface.

## 🧠 Features

- 🔍 Fetches the latest tech news from TechCrunch.
- 🗂️ Categorizes and filters articles based on genre.
- 🧹 Cleans and processes JSON data for structured output.
- 🌐 Serves processed data through a local web app.
